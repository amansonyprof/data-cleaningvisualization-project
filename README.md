# Data Cleaning Project

## Description
This project focuses on cleaning and preprocessing a dataset using Python. The dataset was cleaned by removing missing values, duplicates, and correcting formatting issues to make the data ready for analysis.

## Files Included
- raw_dataset.csv
- cleaned_dataset.csv
- data_cleaning.py

## Cleaning Steps Performed
- Removed missing values
- Removed duplicate records
- Renamed columns properly
- Fixed formatting issues
- Checked data types

## Technologies Used
- Python
- Pandas
- NumPy

## Output
The final cleaned dataset is included in this repository.
